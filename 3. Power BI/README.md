This folder contains the **Power BI dashboard** developed as part of the end-to-end business analytics project.  
The dashboard translates analytical insights into a **management-ready, interactive view** to support data-driven decision-making.

---

## Dashboard Objective

The primary objective of this dashboard is to provide leadership with a **one-page overview** of:

- Overall transaction performance  
- Branch-level contribution and comparison  
- Monthly transaction trends  
- Customer segment contribution  
- Key operational KPIs  

The design focuses on **clarity, consistency, and usability**, rather than visual complexity.

---

## Dashboard Contents

- `dashboard.pbix`  
  Interactive Power BI report containing KPIs, charts, slicers, and tooltips.

- `dashboard_screenshots/`  
  Static images of the dashboard for quick preview and documentation.

---

## Key KPIs Displayed

- **Total Transaction Value**  
- **Total Transactions**  
- **Average Transaction Value**  
- **Active Branches**

These KPIs provide a high-level snapshot of business scale and activity.

---

## Key Visuals & Analysis

- **Branch-wise Transaction Value (Top 10)**  
  Highlights top-performing branches and enables performance comparison.

- **Monthly Transaction Trend**  
  Shows stability and seasonality in transaction activity across the year.

- **Customer Type Contribution**  
  Visualizes the split between Retail and SME customers.

- **Transaction Type Analysis**  
  Compares transaction value across Loan Payments, Deposits, and Withdrawals.

Contextual metrics such as **Average Transaction Value** and **Branch Contribution %** are provided through tooltips to maintain a clean layout while enabling deeper analysis.

---

## Interactivity & Filters

The dashboard includes slicers for:
- Month  
- Region  
- City  

These filters allow users to dynamically explore performance across different dimensions without overwhelming the main view.

---

## Design Principles Followed

- Single-page executive layout  
- Minimal and consistent color palette  
- KPI-first hierarchy  
- Tooltips for secondary insights  
- No unnecessary visuals or clutter  

The dashboard is designed for **business stakeholders**, not technical users.

---

## Intended Audience

This dashboard is suitable for:
- Business leaders and managers  
- Analytics and reporting teams  
- Training and workshop demonstrations  
- Portfolio and interview discussions

## Closing Note

This Power BI report represents the **final visualization layer** of the analytics workflow, building on insights derived from SQL queries and Python-based exploratory analysis.  
The emphasis is on **actionable insights and decision support**, rather than technical complexity.
