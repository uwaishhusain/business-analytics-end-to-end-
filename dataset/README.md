# Dataset – Business Analytics Project

This folder contains the datasets used for the **Python analysis** and **Power BI dashboarding** stages of the Business Analytics project.  
The data represents a **financial services use case** focused on **branch performance, customer behavior, and transaction trends**.

## Files Included
- `branch_data.csv` – Branch and regional information  
- `customer_data.csv` – Customer details and customer type mapped to branches  
- `transaction_data.csv` – Transaction-level financial activity  

## Usage
- **Python**: Data loading, merging, validation, and exploratory data analysis (EDA)  
- **Power BI**: Data modeling, KPI calculation, and interactive dashboards  

## Notes
- The data is **synthetic and anonymized**  
- Created for **learning, training, and demonstration purposes**  
- Structured to reflect **real-world analytics workflows**
